!!! Для удобной смены данных для соединения с базой данных (адрес, пользователь, пароль) можно использовать файл documents/config.txt



!!! Расширение mysqli, использующееся в PHP 7, пока не поддерживает caching_sha2_password аутентификацию, поэтому для работы с mysql 8.0
(для версий ниже 8.0 ничего исправлять не нужно) для пользователя, с которым будет работать скрипт, 
нужно выполнить команду (в mysql клиенте):

ALTER USER 'mysqlUsername'@'localhost' IDENTIFIED WITH mysql_native_password BY 'mysqlUsernamePassword';


Также можно в my.ini поправить default_authentication_plugin на mysql_native_password (но должно помочь и первое).

!!! Используется только фул-дамп таблиц, т.к. импорт дампа Базы данных может породить конфликт привилегий
